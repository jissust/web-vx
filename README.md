# Virtual Xpert — Rediseño y desarrollo web corporativo
Proyecto de rediseño y desarrollo web para Virtual Xpert, enfocado en alinear la presencia digital con la identidad de marca, el brandbook y las propuestas de servicio de la empresa.

Se trabajó sobre una mejora integral del diseño, definiendo una nueva línea visual que represente los valores de la compañía y su posicionamiento como empresa tecnológica. A partir de esta base, se realizó el maquetado completo utilizando HTML, CSS y JavaScript, cuidando la experiencia de usuario y la claridad del mensaje.

Uno de los principales desafíos fue el desarrollo de un formulario conversacional, implementado íntegramente en JavaScript puro, adaptado a la lógica del negocio y a la experiencia que buscaba el cliente.

El proyecto se llevó adelante de forma ágil: el cliente comunicó sus necesidades, se presentó una propuesta de diseño junto con un presupuesto de tiempo y costo, y la implementación se completó en aproximadamente dos semanas. Durante el proceso tuve amplia libertad para proponer ideas, lo que permitió aportar soluciones creativas y mejoras adicionales.

Proyecto finalizado en esta etapa, con posibilidad de futuras iteraciones y mejoras según nuevas necesidades.

Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Diseño responsive
- Desarrollo orientado a performance y UX
